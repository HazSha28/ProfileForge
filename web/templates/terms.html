Description of Service

ProfileForge is a web-based application that transforms heterogeneous candidate data from multiple input sources — including recruiter CSV files, resume PDF documents, and professional platform links — into a single, validated, canonical JSON profile with full provenance tracking and deterministic confidence scoring on every field.

The service includes, but is not limited to:

  • CSV and PDF ingestion and parsing
  • Field normalisation (names, emails, phone numbers, dates, locations, skills)
  • Multi-source merging with configurable priority rules
  • Confidence scoring and provenance attribution per field
  • JSON output with configurable field projection
  • User authentication via email/password, Google OAuth, and GitHub OAuth
